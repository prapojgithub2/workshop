คุณได้รับข้อความใน MS Teams จากหัวหน้าว่า "ฝากเคลียร์ตารางข้อมูลของระบบเก่าๆ ที่ไม่ได้ใช้แล้วออกทีนะ มันทำให้ report query ช้า บางส่วนยังมีข้อมูล 5 ปีที่แล้วค้างอยู่ รีบหน่อยนะ รายละเอียดอยู่ในไฟล์แนบ"

คุณเปิดไฟล์แล้วพบว่ารายละเอียดไม่ชัดเจน ว่าต้องทำอะไรแค่ไหนบ้าง คุณจึงแชตถาม confirm ชื่อ table กลับไปหาหัวหน้า แต่ไม่ได้คำตอบ คุณเห็นว่าระบบทำงานช้าขึ้นเรื่อยๆ กลัวจะไม่ทันการ จึงตัดสินใจ Drop Table ตามที่คิดว่าใช่ — ปรากฏว่าทำให้ข้อมูลลูกค้าบางส่วนหายไป หัวหน้าเรียกคุณไปสอบถามว่าเกิดอะไรขึ้น
🎯 โจทย์
